App starts up
  → RedisConfig creates StringRedisTemplate       (@Bean)
  → SecurityConfig creates BCryptPasswordEncoder  (@Bean)
  → SecurityConfig defines auth rules             (Filter Chain)
  → JwtProperties loads secret + expiry           (@ConfigurationProperties) ← later
  → JwtService reads secret via @Value            (@Value)

Request comes in
  → JwtAuthFilter runs                            (OncePerRequestFilter)
  → Security Filter Chain checks rules            (Filter Chain)
  → Controller handles request


  Register → OTP → verify → set password → login → JWT cookie set → protected endpoints work

  UserAlreadyExistsException thrown in AuthService
  → caught by GlobalExceptionHandler
  → converted to AuthResponse { status: "Failed", msg: "email already exists..." }
  → sent as JSON with HTTP 409
  → frontend reads data.msg
  → showToast(data.msg) displays it to user
